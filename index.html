jquery-a11y-gmap demo

Location : "catonsville, maryland"

This map is showing a 300x225 map of Catonsville, Maryland using a city/state string query separated by comma.

Example Usage : 

var options = {
	width   : 300,
	height  : 225
    };
$( "#map1" ).displayA11yGmap("catonsville, maryland", options);

Location : "42.46441961300047, -87.84984078299965"

This map presents a default sized map of a Latitude/Longitude location using a string query separating the lat and long by comma.

Example Usage : 


/* No Options are set - application will use defaults set on jquery-a11y-gmap.js */

$( "#map2" ).displayA11yGmap("42.46441961300047, -87.84984078299965");

Location : "Des Plaines, IL"

This map presents a 300x225 map of a city/state location. The location string is comma separated as in the first example, however, note that this also takes state abbreviations

Example Usage : 

var options = {
	width   : 300,
	height  : 225
    };
	
$( "#map3" ).displayA11yGmap("Des Plaines, IL", options);

Location : "800 Crystal Lake Blvd, Suite 111 Iron Mountain Mi 49801"

This map presents a larger 400x325 map of a full address location. Additionally, a custom zoom level and marker color are defined in the options here.

Example Usage : 

var options = {
	width        : 400,
	height       : 325,
	marker_color : 'red',
	zoom         : 15
    };

$( "#map4" ).displayA11yGmap("800 Crystal Lake Blvd, Suite 111 Iron Mountain Mi 49801", options);
